# Especificação do Projeto E-Financeiro

## Visão Geral

Projeto de gestão financeira para a ECO-GESTÃO.

## Objetivos

- Gerenciar fluxo de caixa
- Controlar despesas e receitas
- Gerar relatórios financeiros
- Manter backup de dados

## Requisitos Técnicos

- Node.js
- React (Frontend)
- Database (Backend)
